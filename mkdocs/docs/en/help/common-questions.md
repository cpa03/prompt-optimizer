# Common Questions

Frequently asked questions about Prompt Optimizer.

## Getting Started

### What is Prompt Optimizer?

Prompt Optimizer is a tool that helps you improve your AI prompts. It uses AI models to analyze and enhance your prompts, making them more effective for getting better AI responses.

### Do I need an API key?

Yes, you need an API key from at least one AI provider:

- **OpenAI** - Get from [platform.openai.com](https://platform.openai.com)
- **Anthropic (Claude)** - Get from [console.anthropic.com](https://console.anthropic.com)
- **Google (Gemini)** - Get from [ai.google.dev](https://ai.google.dev)
- **DeepSeek** - Get from [platform.deepseek.com](https://platform.deepseek.com)

### Is my data safe?

- All data is stored locally in your browser
- No data is uploaded to our servers
- API calls go directly to your chosen AI provider
- You can export or delete your data at any time

## Usage Questions

### What's the difference between System and User prompt optimization?

- **System Prompt Optimization**: Improves prompts that define the AI's role and behavior (used in chatbots, agents)
- **User Prompt Optimization**: Improves individual user messages to get better responses

### Can I use my own templates?

Yes! You can create custom optimization templates:

1. Go to Template Management
2. Click "Create Template"
3. Define your template structure
4. Save and use for optimization

### How do I save my favorite results?

1. After optimization, click the **Favorite** button
2. Choose or create a category
3. Add relevant tags
4. Access later from the Favorites section

## Technical Questions

### Why am I getting CORS errors?

CORS (Cross-Origin Resource Sharing) errors occur when the browser blocks requests to different domains. Solutions:

1. **Use Desktop App** - No CORS restrictions
2. **Use Chrome Extension** - Bypasses some restrictions
3. **Configure API Provider** - Some providers allow CORS configuration

### Why is optimization slow?

Slow optimization can be caused by:

- **Network latency** - Check your internet connection
- **API rate limits** - Wait and retry
- **Model load** - Try a different model
- **Large prompt size** - Consider splitting into smaller parts

### How do I backup my data?

1. Go to Settings → Data Management
2. Click "Export All Data"
3. Save the JSON file to a safe location
4. Use "Import" to restore from backup

## Pricing & Limits

### Is Prompt Optimizer free?

Yes, Prompt Optimizer is free to use. However, you need:

- Your own API keys from AI providers
- Sufficient API credits/quota from your chosen provider

### Are there any limits?

Limits depend on your AI provider:

- **API rate limits** - Varies by provider and plan
- **Token limits** - Maximum prompt/response length
- **Usage quotas** - Based on your API plan
