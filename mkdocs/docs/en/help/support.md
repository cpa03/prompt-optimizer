### GitHub Issues

For bug reports and feature requests:

1. Visit our [GitHub Issues](https://github.com/linshenkx/prompt-optimizer/issues)
2. Search for existing issues first
3. Create a new issue with:
   - Clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots if applicable
   - Browser/OS version

## When Reporting Issues

Please include:

1. **Environment**:
   - Browser and version
   - Operating system
   - App version (Web/Desktop/Extension)

2. **Problem Description**:
   - What you were trying to do
   - What happened instead
   - Error messages (if any)

3. **Steps to Reproduce**:
   - Detailed steps to trigger the issue
   - Any specific settings or configurations

4. **Screenshots**:
   - Capture any error messages
   - Show the problematic behavior

## Response Time

- **Bug Reports**: Typically within 48 hours
- **Feature Requests**: Reviewed weekly
- **Security Issues**: Immediate priority

## Feature Requests

We welcome suggestions for improvements:

1. Check existing feature requests first
2. Describe the feature clearly
3. Explain the use case and benefits
4. Provide examples if possible
