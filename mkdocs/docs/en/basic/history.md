# History Management

This guide covers how to effectively manage and utilize Prompt Optimizer's history feature.

## 📜 Auto-Save Feature

### Recorded Content

- **Optimization History** - All optimization processes are automatically saved
- **Result Versions** - Each optimization generates a new version record
- **Timestamps** - Precise time of each operation is recorded
- **Model Information** - Model and parameters used are saved

### Data Storage

- All records are stored locally in the browser
- Data is not uploaded to any server
- Supports persistent storage across sessions

## 🔍 History Operations

### Viewing History

1. **Open History**
   - Click the "History" button
   - Or use keyboard shortcuts to access

2. **Browse Records**
   - Sorted in reverse chronological order
   - Displays optimization summary information
   - Supports expanding to view detailed content

### Search and Filter

1. **Keyword Search**
   - Enter keywords in the search box
   - Supports fuzzy matching
   - Real-time filtering of results

2. **Category Filtering**
   - Filter by model type
   - Categorize by optimization mode
   - Filter by time range

### Reusing History Content

1. **One-Click Load**
   - Click a history record to reload
   - Restore the input content from that time
   - Maintain original setting parameters

2. **Copy Results**
   - Quickly copy optimization results
   - Supports copying to clipboard
   - Choose copy format

3. **Continue Optimization**
   - Continue optimization based on history results
   - Maintain continuity of optimization
   - Support iterative improvement

## 🗂️ Record Management

### Organizing Records

1. **Mark Important Records**
   - Add markers to high-quality results
   - Facilitate quick finding later
   - Support custom tags

2. **Group Management**
   - Group by project or theme
   - Create custom categories
   - Support drag-and-drop sorting

### Cleaning Records

1. **Selective Deletion**
   - Select unneeded records
   - Support batch deletion
   - Provides deletion confirmation

2. **Regular Cleaning Recommendations**
   - Clean useless records monthly
   - Retain valuable optimization results
   - Free up local storage space

## 💾 Data Export and Backup

### Export Features

1. **Export Options**
   - Export all history records
   - Export records from specific time periods
   - Export marked important records

2. **Export Formats**
   - JSON format (complete data)
   - CSV format (tabular data)
   - TXT format (plain text)

### Backup Strategy

1. **Regular Backups**
   - Recommend weekly backups
   - Immediately backup important projects
   - Keep multiple versions

2. **Cross-Device Sync**
   - Sync via import/export features
   - Maintain data consistency
   - Support selective synchronization
