# Template Management (Feature Prompts)

This guide covers how to use and manage Prompt Optimizer's optimization template features.

## 📚 Built-in Optimization Templates (Language Switching/Category View)

### General Templates

- **General Prompt Optimization** - Basic optimization suitable for most scenarios
- **Role-Play Optimization** - Specialized for AI character settings
- **Task Instruction Optimization** - Helps AI better understand specific tasks

### Professional Templates

- **Technical Documentation Optimization** - Improves accuracy of technical prompts
- **Creative Writing Optimization** - Stimulates AI creativity
- **Business Communication Optimization** - Suitable for formal business scenarios

### Template Features

- Based on extensive optimization experience
- Optimized for different scenarios
- Continuously updated and improved
- Built-in templates support Chinese/English display switching (name and content change with language)

## ✏️ Custom Templates

### Creating Custom Templates

1. **Create New Template**
   - Click "New Template" or "Add" in the template management interface
   - Set template name and description
   - Select applicable optimization mode (system optimization/user optimization)

2. **Write Template Content**

   ```
   Template Name: [Custom Name]
   Optimization Goal: [Describe the purpose of optimization]
   Optimization Instructions: [Specific optimization rules]
   Applicable Scenarios: [Usage suggestions]
   ```

3. **Test and Refine**
   - Validate using different test cases
   - Adjust template content based on results
   - Document best practices

### Template Management Operations

1. **Edit Template**
   - Modify template name and description
   - Update optimization instruction content
   - Adjust applicable scenario descriptions

2. **Copy Template**
   - Create new version based on existing template
   - Retain original template for modification
   - Suitable for creating similar templates

3. **Delete Template**
   - Remove templates no longer in use
   - Provides deletion confirmation mechanism
   - Support for recovering accidentally deleted templates

## 🔄 Template Usage

### Applying Templates

1. **Select Template**
   - Choose appropriate template in optimization interface (can filter by category)
   - Preview template content
   - Support search and filtering

2. **Template Parameters**
   - Some templates support parameter configuration
   - Adjust optimization intensity
   - Set specific optimization focus areas

### Template Effect Evaluation and Result Display

1. **Effect Comparison**
   - Compare optimization results from different templates (Compare Original vs Optimized)
   - Record applicable scenarios for each template
   - Build personal usage preferences

2. **Continuous Optimization**
   - Improve templates based on usage results
   - Collect user feedback
   - Periodically update template content

3. **Result Display Features**
   - View switching: Source (Markdown source) / Render (rendered)
   - Copy to clipboard: Copy
   - Full-screen view: Fullscreen
   - Expand edit: Expand to enter full-screen editor

## 📤 Import/Export

### Template Sharing

1. **Export Templates**
   - Export custom templates as files
   - Support single or batch export
   - Include complete template information

2. **Import Templates**
   - Import templates from files
   - Support community-shared templates
   - Automatically check template compatibility

### Template Library

1. **Community Templates**
   - Use high-quality templates contributed by the community
   - Periodically update template library content
   - Support ratings and comments

2. **Template Backup**
   - Regularly backup custom templates
   - Prevent accidental loss
   - Support version management

## 💡 Best Practices

### Template Design Principles

1. **Clear Goals** - Each template should have a clear optimization objective
2. **Simple and Efficient** - Avoid overly complex instructions
3. **Reusability** - Design templates with strong versatility
4. **Continuous Improvement** - Constantly optimize based on usage results

### Usage Recommendations

1. **Scenario Matching** - Select the template best suited for the current scenario
2. **Flexible Adjustment** - Fine-tune template parameters based on specific needs
3. **Effect Tracking** - Record usage results of different templates
4. **Experience Summary** - Summarize personal template usage experience
