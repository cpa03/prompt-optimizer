# Quick Start Guide

## Welcome to Prompt Optimizer

Prompt Optimizer is an intelligent tool that helps you transform ordinary prompts into more professional and effective versions.

## 5-Minute Quick Start

### 1. Configure AI Models

1. Click the **Model Management** button in the top right
2. Enter your API keys (OpenAI, Claude, or Gemini)
3. Click **Test Connection** to verify configuration
4. Click **Save** to complete setup

### 2. Start Optimizing Prompts

1. Enter your original prompt in the left input box
2. Select optimization mode (System Prompt / User Prompt)
3. Choose an appropriate optimization template
4. Click **Start Optimization**
5. Wait for AI to generate the optimized result

### 3. View and Use Results

1. View the optimized prompt on the right side
2. Read the reasoning process to understand optimization
3. Click **Favorite** to save quality results
4. Click **Copy** to copy the result
5. Use **Continue Optimization** for further improvements

## Common Questions

### Q: Where do I get API keys?

**A:**

- **OpenAI**: Visit platform.openai.com to register and get API key
- **Claude**: Visit console.anthropic.com to get API key
- **Gemini**: Visit ai.google.dev to get API key

### Q: What if optimization fails?

**A:**

1. Check if network connection is normal
2. Confirm API key is valid and has sufficient quota
3. Try using a different model
4. Simplify the original prompt content

### Q: How to improve optimization quality?

**A:**

1. Provide more detailed original prompts
2. Choose more suitable optimization templates
3. Use iterative optimization for improvements
4. Try different AI models

### Q: Is my data safe?

**A:**

- All data is stored locally in your browser
- Nothing is uploaded to our servers
- You can export or delete data at any time
- API calls connect directly to AI service providers

## Troubleshooting

### Interface Issues

- **Page won't load** - Refresh browser or clear cache
- **Buttons unresponsive** - Check if browser supports modern web standards
- **Display issues** - Try adjusting browser zoom level

### Functionality Issues

- **Optimization timeout** - Check network connection, try again
- **Results not showing** - Wait longer or refresh page
- **History lost** - Check if browser allows local storage

### Performance Issues

- **Slow operation** - Close other browser tabs to free memory
- **High memory usage** - Regularly clean history and cache
- **Response delay** - Check network speed and API service status
