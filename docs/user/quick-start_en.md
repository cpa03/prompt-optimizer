# Quick Start Guide

## 🚀 Welcome to Prompt Optimizer

Prompt Optimizer is an intelligent tool that helps you transform ordinary prompts into more professional and effective versions.

## ⚡ 5-Minute Quick Start

### 1. Configure AI Model

1. Click the **⚙️ Model Management** button in the top right corner
2. Enter your API key (OpenAI, Claude, or Gemini)
3. Click **Test Connection** to ensure the configuration is correct
4. Click **Save** to complete the configuration

### 2. Start Optimizing Prompts

1. Enter your original prompt in the left input box
2. Select optimization mode (System Prompt Optimization / User Prompt Optimization)
3. Choose an appropriate optimization template
4. Click the **Start Optimization** button
5. Wait for the AI to generate optimized results

### 3. View and Use Results

1. View the optimized prompt on the right side
2. Read the reasoning process to understand the optimization principles
3. Click **Favorite** to save high-quality optimization results
4. Click the **Copy** button to copy the result
5. If improvements are needed, use the **Continue Optimization** feature

## ❓ FAQ

### Q: Where can I get API keys?

**A:**

- **OpenAI**: Visit platform.openai.com to register and get an API key
- **Claude**: Visit console.anthropic.com to get an API key
- **Gemini**: Visit ai.google.dev to get an API key

### Q: What should I do if optimization fails?

**A:**

1. Check if network connection is normal
2. Confirm API key is valid and has sufficient quota
3. Try using a different model
4. Simplify the original prompt content

### Q: How to improve optimization quality?

**A:**

1. Provide more detailed original prompts
2. Choose a more suitable optimization template
3. Use the iterative optimization feature for improvement
4. Try different AI models

### Q: Is my data safe?

**A:**

- All data is stored in your local browser
- It will not be uploaded to our servers
- You can export or delete data at any time
- API calls connect directly to AI service providers

## 🔧 Troubleshooting

### Interface Issues

- **Page won't load** - Refresh the browser or clear cache
- **Buttons not responding** - Check if browser supports modern web standards
- **Display issues** - Try adjusting browser zoom level

### Functionality Issues

- **Optimization timeout** - Check network connection, try optimizing again
- **Results not showing** - Wait longer or refresh the page
- **History lost** - Check if browser allows local storage

### Performance Issues

- **Slow operation** - Close other browser tabs to free memory
- **High memory usage** - Regularly clean up history and cache
- **Response delay** - Check network speed and API service status
